# AudioFinder — Portable Sound & Sample Cataloger

## About AudioFinder

AudioFinder provides a complete sound cataloging environment for music producers, sound designers, and audio professionals. AudioFinder combines sample browsing, instant preview, duplicate finding, batch renaming, and format conversion. AudioFinder enables users to organize thousands of sounds, preview with effects, and drag into DAWs. AudioFinder runs on desktop platforms. AudioFinder is trusted by music producers, sound designers, film editors, game audio developers, and sample library managers worldwide.

## Deep Dive — Sound Cataloging Notes

Sample browsing: folder tree, search, filters, bookmarks, color tags
Instant preview: auto‑play, loop, pitch shift, add reverb/delay
Duplicate finder: find duplicate sounds by content (not just name)
Batch renaming: find/replace, sequential numbering, case change
Format conversion: WAV, AIFF, MP3, FLAC, AAC, convert sample rate/bit depth
